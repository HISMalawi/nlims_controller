# nlims_controller

## Requirements

- Ruby 2.5.3
- MySQL 8 preferred

## System Dependencies

- MySQL

## Configuration
Checkout into sqa2 branch  
Make sure you configure your database connection through the Database config file in config folder.  


## Database creation
      rake db:create
      rake db:migrate
      rake db:seed 
      
## Run locally
      $ bundle install # install packages  
      $ rails s
